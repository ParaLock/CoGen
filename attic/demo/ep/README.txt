This directory will contain the synthesized variations.

Note that the junit.jar file is a release of JUnit 4.11

The packages.zip file contains the generated code for six different EP approaches for model M8,
which contains a wide range of datatypes and operations.

To generate all variations, do the following steps:

1. Within sbt shell, execute "language-java/run"

2. Execute TestSynthesis (in language/java/src/main/java/buildTest/TestSynthesis
